## Qué construyes

Un SaaS donde los usuarios pagan suscripción mensual para acceder a tu producto. Stripe maneja el cobro; tu app desbloquea features según el plan de cada usuario.

## Pasos

1. Sigue [Stripe paso a paso](/docs/setup/stripe): crear la cuenta, el producto con su precio, y copiar tus keys a `web/.env.local`. Ahí mismo activas `features.payments: true` y pegas el `stripePriceId` en tu plan de `web/config.js`.
2. Deja que tu IA construya el checkout y el webhook con el prompt del **Paso 6** de esa guía (un **webhook** es una URL de tu app que otro servicio — aquí Stripe — llama automáticamente cuando pasa algo, por ejemplo un pago exitoso).
3. Cuando tengas deploy, registra el webhook en Stripe (**Paso 7** de la guía).
4. Prueba el flujo completo con la tarjeta de prueba `4242 4242 4242 4242`: pagar → volver a tu app → ver las features de plan Pro desbloqueadas.
