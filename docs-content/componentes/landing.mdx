Los componentes en `web/components/landing/` están diseñados para **no tocarlos**. Cambias el copy en `web/config.js` y el componente se actualiza solo.

| Componente | Lee de config | Cuándo personalizar |
|---|---|---|
| `Hero` | `landing.hero` | Casi nunca — cambias copy en config |
| `Features` | `landing.features` | Cambias los items en config; cambias el componente si quieres otro layout |
| `FAQ` | `landing.faq` | Igual que Features |
| `Waitlist` | `landing.waitlist` | Solo si quieres cambiar el formulario (campos, validación) |

## Para agregar una sección nueva

1. Crea el componente en `web/components/landing/MiSeccion.js`.
2. Lee el copy de `config.landing.miSeccion`.
3. Agrega `miSeccion: {...}` en `web/config.js`.
4. Importa y renderiza en `web/app/(marketing)/page.js`.
