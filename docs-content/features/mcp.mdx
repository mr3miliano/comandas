**MCP** (Model Context Protocol) es un estándar para que las apps de IA se conecten entre sí: le da a una IA una forma universal de usar herramientas de otras apps, como un "USB" para inteligencia artificial.

## Doble rol (en construcción)

Cuando esta feature esté lista en el boilerplate:

- **Servidor MCP** — vivirá en `/api/mcp` y expondrá el mismo registry de tools de `web/lib/tools/index.js`. Cualquier cliente MCP (Claude Desktop, otros) podrá usar tu producto como herramienta.
- **Cliente MCP** — vivirá en `web/lib/mcp/client.js` y consumirá MCPs externos (Gmail, Notion, ESP-Claw).

Será la pieza central que conecta Web ↔ Hardware en las Semanas 8–9.
