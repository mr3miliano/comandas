<Callout type="info">
Esta feature se activa en las **Semanas 8–9** del curso.
</Callout>

Activación: `features.hardware: true` en `web/config.js`. Los ejemplos de firmware se publicarán en la carpeta `firmware/` antes de la Semana 8 (hoy la carpeta está vacía, en construcción).

Tu web app no necesitará cambios: cuando el servidor MCP del boilerplate esté listo (vivirá en `/api/mcp` — ver [MCP](/docs/features/mcp)), el ESP32 se conectará a él como cualquier otro cliente MCP.
