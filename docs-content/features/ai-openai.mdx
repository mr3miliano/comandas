## Helpers principales

Viven en `web/lib/openai/`:

```js
import { streamChat } from "@/lib/openai/chat"
import { generateObject } from "@/lib/openai/structured"
```

- `streamChat(messages)` — chat streaming para UIs interactivas (lo usa `/api/ai/chat`).
- `generateObject(schema, prompt)` — JSON tipado garantizado por el modelo (lo usa `/api/ai/structured`).

Configuración de modelos y costo en la sección `ai` de `web/config.js`.
