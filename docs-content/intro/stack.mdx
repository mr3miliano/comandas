VibeFast tiene **una decisión por capa**. No comparas opciones — usas lo que viene y te enfocas en tu producto. Cuando termines el curso, ya sabrás suficiente para cambiar lo que quieras.

## Capa por capa

| Capa | Herramienta | Por qué |
|---|---|---|
| Framework web | **Next.js 15** (App Router, JavaScript) | El framework más popular para apps full-stack con React. JavaScript (no TypeScript) para reducir fricción inicial. |
| Estilos | **Tailwind 4** + **DaisyUI** | Tailwind para utilidades, DaisyUI para componentes pre-hechos bonitos. No más CSS desde cero. |
| Base de datos | **Supabase** (Postgres + pgvector) | Postgres serio + interfaz amigable + auth incluida + vector search para RAG. |
| Auth | **Supabase Auth** con Google | Login con un click, sin manejar passwords. |
| Email | **Resend** + React Email | Templates en React, envío con API limpia. |
| IA | **OpenAI SDK** + Zod (structured outputs) | El modelo más maduro para tool use y agentes. |
| Agentes | **LangGraph.js** | Framework para agentes con grafos (percibe → decide → actúa). |
| Conexión a herramientas | **MCP** (Model Context Protocol) | Estándar abierto para conectar IA con herramientas y hardware. |
| Analytics | **PostHog** (opcional) | Tracking de eventos sin pelear con Google Analytics. |
| Hosting | **Vercel** | Deploy con `git push`. Versión gratis suficiente para Demo Day. |
| Package manager | **yarn** 1.x (classic) con workspaces | El que pide el temario. |

## Lo que NO incluye (a propósito)

- **TypeScript** — el temario es explícito en JavaScript. Menos sintaxis, menos atorones. Puedes migrar después.
- **shadcn/ui** — DaisyUI cubre el mismo terreno con menos pasos.
- **Next-Auth / Clerk** — Supabase Auth es suficiente y mantiene una sola fuente de verdad.
- **Drizzle / Prisma** — Supabase client + SQL directo es más legible para principiantes.
- **Stripe** — opcional, detrás de `features.payments` en `config.js`.

## Costos esperados

Con la versión gratis de todo, **un MVP del curso cuesta ~US$5–20** — en total durante el curso, la mayoría en créditos de OpenAI (con `gpt-4o-mini` como default). Vercel y Supabase: gratis hasta cientos de usuarios.
